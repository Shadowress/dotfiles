# Dotfiles

This repository contains personal dotfiles for configuring and setting up development environments across multiple machines and operating systems.

---

## Installation

To set up your dotfiles on a new machine:

1. Clone the repository into your home directory:

```bash
git clone https://github.com/Shadowress/dotfiles.git .dotfiles
```

2. Navigate into the repository:

```bash
cd .dotfiles
```

3. Run the install script:

```bash
bash install.sh
```
⚠️ Note for Windows users:
PowerShell and Command Prompt do not support Bash scripts.
Please use Git Bash (installed automatically with Git for Windows) to run the command.
